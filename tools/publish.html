Публикация отзыва

Вставьте полученный отзыв целиком как скопировали — ничего не вырезайте и не правьте.

  1. Скопируйте блок вида review:id {"id":"…",…}
  2. Вставьте ниже → «Сделать JSON»
  3. Объект → в data/reviews.json
  4. id → в data/published.json
  5. Ссылка → страница otziv.html?id=…

Вставьте объект в массив [...]. Поле name — заголовок на otziv.html; ссылка строится как ?id=cursor из name.
