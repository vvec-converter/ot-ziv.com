Публикация отзыва

Только для вас. Скопируйте одну строку из Google Таблицы → вставьте → получите JSON для GitHub.

  1. В таблице выделите строку отзыва (не заголовок) → Ctrl+C
  2. Вставьте ниже → «Сделать JSON»
  3. Скопируйте результат в data/reviews.json и id в data/published.json на GitHub

В published.json массив id: добавьте в кавычках через запятую.
